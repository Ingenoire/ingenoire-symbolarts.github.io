The video focuses on the teleporter to the arena, the countdown timer, the targets, the counter that checks the amount of targets broken, and the logic to open the door. This isn't the entire code of the actual level as it is now, just the basics that were used to create the system of Break the Targets.
  • If you wish to add more targets, make sure to increase the counter's required number to send a signal. And don't forget to implement the same connect settings for each new target.
  • The connect ID of 1 is used as both a reset and as a start for the game logic: if someone enters the teleporter, it both resets all items, and also starts the countdown.
  • Always code your connections BEFORE you start building a stage.

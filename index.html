To install Symbol Arts, simply drop the .SAR file into your Documents\SEGA\PHANTASYSTARONLINE2_NA_STEAM\symbolarts\import (we're assuming you use Steam), then open PSO2, go to Symbol Arts, set the current tab to Import, and press Import. Your SA should appear in it. Press save and override a slot of your SA palette, and you should be able to use it.

Using this online symbol art editor, you can open the .saml files for better editing, especially since some objects are grouped into easier to manipulate folders.
